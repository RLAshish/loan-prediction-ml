# Loan Prediction Machine Learning Project

## Overview

This project predicts loan approval status using multiple machine learning algorithms.

## Features

* Data preprocessing & cleaning
* Feature engineering
* Model training with 8+ algorithms
* Model comparison using ROC-AUC

## Algorithms Used

* Logistic Regression
* KNN
* SVM
* Decision Tree
* Random Forest
* AdaBoost
* Gradient Boosting
* XGBoost

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

##  Result

Best performing model selected based on ROC-AUC score.
